## Cryptography

- [Nettle.jl](https://github.com/JuliaCrypto/Nettle.jl) - Libnettle cryptographic primitives bindings.
- [SHA.jl](https://github.com/JuliaCrypto/SHA.jl) - Performant SHA hash functions.
- [MD5.jl](https://github.com/JuliaCrypto/MD5.jl) - MD5 hash implementation.
- [MbedTLS.jl](https://github.com/JuliaLang/MbedTLS.jl) - TLS and SSL protocol implementation.

## Network Security

- [HTTP.jl](https://github.com/JuliaWeb/HTTP.jl) - HTTP client and server framework.
- [Sockets.jl](https://docs.julialang.org/en/v1/stdlib/Sockets/) - Low-level networking primitives.
- [LibSSH2.jl](https://github.com/JuliaCloud/LibSSH2.jl) - SSH protocol bindings.

## Binary Analysis

- [DWARF.jl](https://github.com/JuliaDebug/DWARF.jl) - DWARF debug information parser.
- [ELF.jl](https://github.com/JuliaInterop/ELF.jl) - ELF binary format parser.
- [MachO.jl](https://github.com/JuliaInterop/MachO.jl) - Mach-O binary format parser.

## Data and Forensics

- [CSV.jl](https://github.com/JuliaData/CSV.jl) - Fast CSV file parser.
- [JSON3.jl](https://github.com/quinnj/JSON3.jl) - High-performance JSON parser.
- [DataFrames.jl](https://github.com/JuliaData/DataFrames.jl) - Tabular data analysis framework.

## Machine Learning Security

- [Flux.jl](https://github.com/FluxML/Flux.jl) - Machine learning framework for anomaly detection.
- [MLJ.jl](https://github.com/alan-turing-institute/MLJ.jl) - Machine learning toolbox for threat classification.
- [Knet.jl](https://github.com/denizyuret/Knet.jl) - Deep learning framework.
