# Contributing

## Adding a Resource

1. Fork the repository
2. Add your resource to the appropriate category in `readme.md`
3. Follow the format: `- [Name](URL) - Brief description.`
4. Submit a pull request

## Guidelines

- Resource must be related to Julia and security/cryptography
- Must have a working URL
- Must have a meaningful description
- No duplicates
- Alphabetical order within categories

## Awesome List Standards

This list follows the [awesome-lint](https://github.com/sindresorhus/awesome-lint) standards:
- No badges in header
- No center-aligned text
- List format (not tables)
- Lowercase `readme.md`
- Contents section required
