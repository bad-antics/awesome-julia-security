# Contributing to Awesome Julia Security

Thank you for your interest in contributing!

## Guidelines

### Adding a Package

1. Make sure the package is related to security, cryptography, or related fields
2. The package should be actively maintained (commits within the last year)
3. Add your package in alphabetical order within its category
4. Use the following format:

```markdown
- [Package Name](https://github.com/user/package) - Short description.
```

### Quality Standards

- Package should have documentation
- Package should have tests
- Package should be installable via Pkg

### Submitting

1. Fork this repository
2. Create a new branch: `git checkout -b add-package-name`
3. Add your changes
4. Submit a pull request

## Code of Conduct

Be respectful and professional. This is a security-focused list, so ensure packages are legitimate and not malicious.
